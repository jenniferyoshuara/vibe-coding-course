# Vibe Coding Operator Course

An interactive, self-contained web course that teaches you **enough coding to safely operate AI-built projects** — reviewing AI changes, running tests, protecting API keys, and shipping analyst-grade output without blindly trusting the AI.

It's built around a real use case: running and supervising a stock-research bot.

## What's inside

A single HTML file ([`index.html`](index.html)) — no build step, no dependencies. Just open it in a browser.

- **8 weekly lessons** with real teaching content (not just a checklist): the terminal & project anatomy, Git review, reading Python, config & secrets, testing & debugging, JSON/schemas/Pydantic, Markdown content generation, and operating discipline.
- **Embedded explanations**, syntax-highlighted code examples, and "key idea" / "watch out" callouts.
- **Interactive activities** after every section: multiple-choice quizzes with instant feedback, plus "build the command" exercises.
- **5 applied projects**, each with its own quiz.
- **Progress tracking** (saved in your browser via `localStorage`), search, topic/week filters, a checklist view, cheat sheets, and print-to-PDF.

## How to use it

1. Download or clone this repo.
2. Open `index.html` in any modern browser.
3. Work through the weeks. Your progress saves automatically on that device.

## Live version

If GitHub Pages is enabled for this repo, the course is served at the Pages URL (Settings → Pages → deploy from the `main` branch).
